# Excluir Emails Gmail

Este script em Python permite excluir emails da pasta "INBOX" do Gmail com base em palavras-chave no assunto. Ele usa o protocolo IMAP para conectar à sua conta do Gmail, buscar os emails e removê-los fisicamente após marcação.

## Funcionalidades

- Exclui emails de acordo com palavras-chave no assunto.
- Usa o protocolo IMAP para conectar ao Gmail.
- Permite excluir todos os emails ou filtrar por palavras específicas.
